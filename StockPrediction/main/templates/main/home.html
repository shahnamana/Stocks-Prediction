About the Project

We try to develop a web application where a user can come on our portal and see the live stock price from our list of stocks. Our portal also provides a functionality for a registered user to predict price of their stock from their portfolio.

For the web app we have used Django as the web framework for python for designing all the front end and backend of the website.

For Machine Learning Model for future price prediction, we have use ARIMA model. ARIMA is a preferred algorithm when it comes to predicting any future values i.e., forecasting the values. With the help of python, we pre-process and clean the data and obtain the final dataset which is our target value and one responsible for forecasting the future values. ARIMA algorithm uses two techniques , Auto Regressor and Moving Average, which are integrated together and used to forecast the value for the future price.
